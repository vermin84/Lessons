Побудова кота звичайного

  • Мурчало
  • Гарчало
  • Стрибало
  • Карябало
  • Махало
  • Гризло

Мурчало

Мурчало знаходиться всередені кота звичайного. Воно призначене для прояву позитивних емоцій кота. Має дуже заспокійливі властивості

Гарчало

Гарчало знаходиться неподалік від мурчала. Воно призначене для прояву незадоволеності та негативних емоцій кота. Має дуже незаспокійливі властивості

Стрибало

Стрибало знаходиться знизу кота. Призначене для стрибання кота на велику висоту, та для наведення суцільного безладу в приміщенні

Карябало

Карябало знаходиться наприкінці стрибала. Призначене для лазіння по шторах, та агресивних нападів на кінцівки господарів

Махало

Махало знаходиться на задній частині кота, призначенне для індикаціі нервогого стану кота, а також для рульожки під час стрибків

Гризло

Гризло знаходиться на передній частині кота. Воно призначене для годування кота, а також для гризення кінцівок господаря, або кабелів живлення всілякіх гаджетів
